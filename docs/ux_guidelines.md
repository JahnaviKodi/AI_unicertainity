# UX Guidelines — AI Uncertainty Handling

## Response Colour Coding
- GREEN  — ANSWER (confident response)
- AMBER  — CLARIFY (needs more information)
- RED    — ABSTAIN (not enough information)
- PURPLE — ESCALATE (high risk, consult expert)

## When to Show Uncertainty Messages
- Always show uncertainty level alongside the response
- Never hide abstention decisions from the user
- Always provide a next step (clarify, escalate, retry)

## User Feedback
- Show thumbs up / thumbs down on every response
- Ask user if the abstention was correct or not
- Use feedback to retrain thresholds over time

## Escalation Pathways
- Medical queries → Recommend a licensed doctor
- Legal queries → Recommend a qualified lawyer
- Financial queries → Recommend a certified advisor

## Accessibility
- Use icons alongside colour coding for colour blind users
- Ensure screen readers can read uncertainty messages
- Keep uncertainty language simple and jargon free
