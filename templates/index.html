📄 API Documentation
REST API endpoints for the AI Uncertainty System.

POST  /ask

Submit a query to the AI uncertainty pipeline.

FieldTypeDescription
querystringThe question to ask the AI
Request: {"query": "How does ML work?"}

Response: {
  "decision": "ANSWER",
  "confidence": 0.8,
  "boundary": "IN_DISTRIBUTION",
  "response": "...",
  "response_time": 0.02,
  "id": 1
}

POST  /upload

Upload a .txt, .pdf, or .docx file for analysis.

FieldTypeDescription
filefileThe file to upload (.txt .pdf .docx)

POST  /ask-with-file

Ask a question about an uploaded file.

FieldTypeDescription
querystringQuestion about the file
file_contentstringExtracted text from the file

POST  /feedback

Submit feedback for a response.

FieldTypeDescription
idintegerQuery ID
feedbackstring"up" or "down"

GET  /analytics

Get query counts per decision type.

{"total":10,"ANSWER":5,"CLARIFY":2,"ABSTAIN":2,"ESCALATE":1}

GET  /history

Get full query history from the database.

GET  /export-csv

Download full query history as a CSV file.
